# Smart Parking Management System

## 📌 Project Description
The **Smart Parking Management System** is a web-based application developed to help drivers easily find, book, and navigate to available parking spaces. The system enables parking owners to manage their parking facilities and allows a Super Admin to control approvals and monitor overall system operations. Drivers can make reservations and view directions to their selected parking spaces using integrated map services.

---

## 🛠️ Technology Stack
- **Backend:** Java Spring Boot  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** MySQL  
- **Mapping/Navigation:** Google Maps (URL integration)

## ⚙️ Features

### Driver
- Search and select parking spaces manually
- Make online payments
- Get directions to selected parking spaces
- View reservation history

### Parking Owner
- Register parking facilities
- Manage parking slots and availability
- Monitor reservations
- Track earnings

### Super Admin
- Approve or reject parking owner registrations
- Monitor system activity
- View reports and analytics
- Manage users
